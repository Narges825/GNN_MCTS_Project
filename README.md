# GNN_MCTS_Project
Solving the Traveling Salesman Problem Using Graph Neural Networks

The project consists of two parts:

## Part A – Google Colab
1) Upload the .ipynb Notebook file in Google Colab
2) Upload the .zip Folders from the local Content Folder into Google Colab Content Folder
3) Run the Notebook
4) Download result.csv and test_tsp.zip
5) Go to PartB - Local Solver

 ## Part B – Local Solver
 Python version 3.9
 Gurobi version: 12.0.1
 Gurobi requires a valid license.   

1. Save first 50 tsp instances in test_tsp file
2. Run each solver 
3. Add csv data to results
